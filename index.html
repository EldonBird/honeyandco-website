About

The primary goal of this project is to connect apiarists together to allow for more information to be shared about their hives. This information should help apirists optimize and improve the quality of their hives through the use of collectivised testing and information gathering. The more data that is added to this the better and should help in the future with planning where and how to host your hives. This information is also accessible to anyone, I just ask that if you are getting any value that you please contribute!
